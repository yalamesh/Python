In python we are not required to specify the type explicity.Based on value provided,the type will be asigned automatically.hence python is dynamically typed language.

2.float
3.complex

7.bytearray

9.list

int data type:

we can use int data type to represent whole number(integral values)
Ex: a=10
	type(a)#int
we can represent  int values in the following ways.
1.decimal form
2.Binary form
3.octal form
4.Hexa decimal form

1.Decimal form(base-10):
It is the default number system in python
The allowed digits are: 0 to 9
Ex: a=10

2.Binary form(base-2):
The allowed digits are : 0 & 1
Literal values should be prefixed with ob or OB
EX: a=OB1111
    a=OB123
    a=b111
Octal Form(base-8):
	
The allowed digits  are :0 to 7
Literal value should be prefixed with oO or OO
Ex:a=Oo123
   a=Oo786
